# Курсовая работа "Игра в Каллах"
Это репозиторий для курсовой работы по основам программирования "Игра в Каллах".

### Структура репозитория

- В корневой директории находится `.pdf` файл отчёта и этот README файл.
- В папке *DocX* находится файл отчёта в формате `.docx`.
- В папке *LaTeX* находится `.pdf` файл, скомпилированный их LaTeX.
- В папке *Программа* находится исполняемый `.exe` файл программы. В подпапке *Исходники* находятся файлы с исходным кодом, а именно:
    1. *game.h*
    2. *game.cpp*
    3. *main.cpp*
